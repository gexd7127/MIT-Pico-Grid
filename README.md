# MIT PicoGrid
The goal of this project is to develop controller firmware for inverter-based microgrids.

## List of Components/Tools:

- TI C28379 microcontroller/TI CCS 6.0 IDE
- ST STEVAL-IHM028V2 2kW 3-phase IGBT inverters
- LabVIEW 14.0
